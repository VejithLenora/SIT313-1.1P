Here is some of my work:

ArduRover

The ArduRover: The ardurover is a rover which operates on an Arduino Uno. It is a rover whose main purpose is to capture surveillance. The device is made up of a Arduino Uno, Motors, Motordriver, Bluetooth Communication Unit and an ESP32Cam which allows video footage to be captured and can be streamed on a different platform.

RaspCam:

The RaspCam was a very simple project I created for the unit SIT210. This device operated using a Arduino Uno, A PIR Motion Sensor and a ESP32Cam. The RaspCam is a camera that takes pictures whenever motion is detected. This allows the RaspCam to have basic security functionality at a fraction of the cost of most security cameras and it also uses much less storage.
